# A/B Testing & Regression Analysis – Marketing Campaign Performance

End-to-end statistical analysis comparing Facebook Ads and Google AdWords using A/B testing and regression modeling. The project evaluates platform effectiveness, identifies statistically significant performance differences, and predicts conversions to support data-driven marketing decisions.



## ❓ Business Problem

Marketing teams invest heavily in digital advertising platforms but often lack statistical validation when deciding where to allocate budgets.

Key challenges include:

* Uncertainty about which platform drives higher conversions
* Variability in campaign performance
* Lack of data-backed decision-making
* Difficulty predicting conversion outcomes

This project analyzes campaign performance using A/B testing and regression to determine the most effective platform and optimize marketing spend.



## 🎯 Project Objectives

* Compare Facebook vs Google AdWords performance
* Conduct A/B hypothesis testing for statistical significance
* Analyze conversion and cost trends
* Build regression models to predict conversions
* Generate actionable business insights



## 🧠 Key Business Questions Answered

* Which advertising platform performs better?
* Are performance differences statistically significant?
* How does ad spend influence conversions?
* What trends emerge across time?
* Which platform should receive higher budget allocation?



## 🗂️ Dataset Overview

Synthetic marketing dataset simulating real-world campaign performance:

* Daily ad views, clicks, conversions, and spend
* CTR, CPC, and conversion rate metrics
* Platform comparison across a full campaign year

📌 Designed to mirror real advertising analytics scenarios.
